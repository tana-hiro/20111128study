チェックアウト

gitのチェックアウトに関する機能(SVN的な意味合い)

作成

既存のデータから作成する

cd ~/projects/myproject
git init
git add .

既存のリポジトリから作成する

git clone ~/existing/repo ~/new/repo
git clone git://host.org/project.git
git clone ssh://you@host.org/proj.git

更新

originから最新の変更を変更する

git fetch

(マージはしない)

originから最新の変更を取得する

git pull

(取得した後、マージする)

他人から届いたパッチを当てる

git am -3 patch.mbox

(競合発生時は、git am -resolved で解消する git am --resolved )
